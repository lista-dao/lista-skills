# Reference Index

Files in `references/` contain detailed instructions and templates for each report type. Read only the files needed for the selected report.

## references/computation.md
Shared metric computation: price fetching (ERC20/LP/fallback), LTV, health factor, liquidation price, correlated pair detection, risk level assignment.
→ Read when generating: Position Report (1), Risk Check (4), Daily Digest (5).

## references/position.md
Report A — Position Report. Fetch positions, compute metrics, generate recommendations, EN + 繁體中文 templates.
→ Read when user selects report type 1.

## references/market.md
Report B — Market Lending Rates. Per-market Supply APY, Borrow APY, liquidity, utilization. Supports asset filtering. EN + 繁體中文 templates.
→ Read when user selects report type 2.

## references/yield.md
Report C — Vault Yield. Per-vault APY (base + LISTA), TVL, underlying market allocations. Grouped by zone. EN + 繁體中文 templates.
→ Read when user selects report type 3.

## references/risk.md
Report D — Risk Check. Alert thresholds, EN + 繁體中文 templates (alert + all-clear), threshold customization, push notification setup.
→ Read when user selects report type 4.

## references/digest.md
Report E — Daily Digest. Positions + yield estimation + market snapshot, EN + 繁體中文 templates, subscription setup.
→ Read when user selects report type 5.
